# Example Initiative — One Click Checkout

## Что это

Инициатива по ускорению оформления заказа с уменьшением числа шагов.

## Задачи

- сократить время до подтверждения заказа;
- снизить число брошенных checkout flows;
- не нарушить инварианты pricing и fraud checks.

## Затронутые зоны

- checkout domain
- pricing engine
- auth / session handling
- analytics / metrics

## Основные риски

- скрытые зависимости в монолите;
- разъезд логики расчёта;
- нарушение ранее неявных инвариантов.
