# Example Evidence Bucket — PRs

Здесь можно хранить ссылки или описания pull request-ов, связанных с конкретными инициативами и зонами ответственности.

## Задачи

- подтверждать фактическое изменение в коде;
- связывать implementation с инициативой или решением;
- показывать reviewer trail и историю обсуждения.
