# Example Capability — Identity Core Team

## Назначение

Команда и подсистема, отвечающая за идентичность пользователя, аутентификацию и жизненный цикл сессий.

## Задачи уровня

- удерживать security invariants;
- поддерживать стабильность session lifecycle;
- публиковать изменения, влияющие на вход и авторизацию;
- давать соседним зонам понятный и актуальный контекст.
