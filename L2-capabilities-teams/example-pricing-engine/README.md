# Example Capability — Pricing Engine Team

## Назначение

Команда и подсистема, отвечающая за вычисление цены, скидок и правил расчёта стоимости.

## Задачи уровня

- держать корректность расчётов;
- обеспечивать прозрачность правил ценообразования;
- минимизировать каскадные эффекты при изменении логики;
- публиковать инварианты для соседних зон.

## Критичные инварианты

- цена должна быть детерминируемой для одного и того же контекста;
- изменения discount logic должны быть обратимо трассируемы;
- интеграции с checkout не должны ломать финальный расчёт.
