# Example Domain — Checkout

## Назначение

Пример домена для оформления покупки и подтверждения заказа.

## Задачи домена

- минимизировать трение при оплате;
- обеспечивать корректное прохождение заказа;
- удерживать стабильность интеграций с платёжной логикой;
- координировать изменения между frontend, backend и anti-fraud.
