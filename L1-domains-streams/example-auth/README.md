# Example Domain — Auth

## Назначение

Пример домена для аутентификации, сессий и управления идентичностью пользователя.

## Задачи домена

- обеспечивать безопасный и быстрый вход;
- снижать число сбоев аутентификации;
- балансировать UX и security;
- координировать изменения между core identity, session management и customer support.
