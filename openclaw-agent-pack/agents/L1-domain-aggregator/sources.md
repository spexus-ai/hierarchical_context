# L1 Sources

Перед запуском `L1 Domain Aggregator` сначала прочитай этот файл.

## Назначение

Этот файл задаёт, какие источники используются для доменной агрегации.

## Primary Inputs

- `L2` контексты домена
- project map внутри домена

## Supporting Inputs

- релевантные `L3` streams для уточнения active coordination
- strategic goals, если домен вносит прямой вклад в `L0`

## Aggregation Rules

- `L1` агрегирует только из уже собранных `L2`
- `L1` не должен напрямую подменять собой `L3` или `L4`
- если нужен факт, спускайся вниз по down-links

## Cross-project Rules

- `L1` обязан показывать cross-project flows внутри домена
- `L1` обязан отмечать knowledge transfers между проектами, если они реально обнаружены
- один домен может содержать несколько проектов портфеля
