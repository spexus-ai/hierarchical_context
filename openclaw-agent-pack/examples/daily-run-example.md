# Daily Run Example

## Вход
- новые PR
- обновления метрик
- один инцидент

## Ожидаемый проход
1. L4 извлекает факты
2. L3 создаёт/обновляет карточки изменений
3. L2 обновляет affected team contexts
4. L1 пересобирает affected domains
5. L0 обновляет global picture
6. OpenClaw получает задачу из L3 и публикует evidence обратно в L4
